# C++ Banking System

## Overview

This is a console-based banking system written in C++. It allows users to perform basic banking operations while practicing fundamental C++ programming concepts.


## Features

- Deposit money
- Withdraw money
- Transfer money
- Check account balance
- Input validation
- Confirmation before withdrawals and transfers
- Menu-driven interface


## Concepts Practiced

- Functions
- Parameters and reference parameters
- Variables
- While loops
- If / Else statements
- Input validation
- User interaction
- Git & GitHub


## Future Improvements

- PIN authentication
- Multiple accounts
- Transaction history
- Save data to files
- Interest calculation
